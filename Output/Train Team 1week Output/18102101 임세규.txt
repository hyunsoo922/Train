- 사용언어 및 수준
	C / C++ / JAVA / 쉘 / HTML5 / CSS3 / JavaScript
- OS / DBMS / 프레임워크
	윈도우 / MySQL / 스프링 프레임워크
- 사용가능 개발툴
	Visual Studio, Visual Studio Code, VMware Workstation, Eclipse, Arduino

- 프로젝트 경험
	1. 체스 프로그램
	- Visual Studio를 이용하여 체스 프로그램을 제작

	2. 스노트 기반의 침입방지시스템
	- VMware Workstation을 이용하여 쉘 프로그래밍으로 외부에서의 침입방지시스템을 제작
	- 기업의 서버, PC, ips(침입방지시스템)를 내부 인트라넷에 구성된 SERVER-FARM으로 구현
	- 내부망 바깥에 있는 PC를 홈 네트워크로 구현하여 라우터를 ips에 연결하고 SERVER-FARM에 접속하기 위해
	  VPN 터널을 구성
	- ips에 보안정책을 수립하여 특정 대역의 ip 주소를 감시, 차단(DROP)

- 프로젝트 멘토기업 선정 및 제출
	퓨처누리, 에프원시큐리티
